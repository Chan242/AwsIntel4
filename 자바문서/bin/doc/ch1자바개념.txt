자바(Java Programming Language)

*자바 언어의 특징
-운영체제에 독립적이다
자바로 작성된 프로그램은 운영체제와 하드웨어에 관계없이
실행가능하다
-객체지향언어이다
상속, 캡슐화, 다형성 등
-비교적 배우기 쉽다

-자동 메모리 관리(Garbage Collection)
가비지 컬렉터가 자동적으로 메모리를 관리해주기 때문에
프로그래머는 메모리를 따로 관리하지 않아도 된다?
왠만하면 프로그래머가 프로그래밍에 집중할 수 있도록 도와준다
-동적 로딩(Dynamic Loading)을 지원한다
실행 시에 모든 클래스가 로딩되지 않고 
필요한 시점에 클래스를 로딩하여 사용할 수 있다

*JVM(Java Virtual Machine)
자바를 실행하기 위한 가상 컴퓨터

JDK(Java Development Kit): 자바로 개발을 하는 프로그래머들이 설치하는 개발 도구

JRE(Java Runtime Enviroment): 개발된 자바 프로그램을 실행만 하면 되는 
평펌한 일반 사용자들이 설치하는 환경

JDK를 설치하면 내부에 JRE도 포함되어 있다.

*컴파일러(Compiler)
사람이 읽을 수 있는 고급 언어로 작성된 코드를 컴퓨터가 
이해할 수 있는 저급 언어(기계어)로 변환해 주는 다리 역할

IDE(Itegrated development editor or environment)
통합 개발 편집기, 환경

*주석(comment)
작성하는 프로그램의 크기가 커질수록 프로그램을 이해하고 변경하는 일이 
점점 어려워진다. 내가 작성했든 남이 작성했든 이해를 돕기 위해 
사용하는 것
주석은 프로그램의 작성자, 작성일시, 버전과 그에 따른 변경이력 등의
정보를 제공할 목적으로 사용된다
범위 주석: /*  수많은 코드나 내용들  */ 기호 사이의 글자만 주석으로 간주한다
한줄 주석: // 한 줄까지의 내용을 주석으로 간주한다
